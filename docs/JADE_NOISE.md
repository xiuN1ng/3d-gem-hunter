# 天然玉石切面噪声方案

## 原方案

切面并不是静态图片：`JadeVolume` 在切割平面上逐像素采样 3D 体积场，再把种水、颜色、棉与裂映射为 HSL/RGBA。原实现主要使用三次插值的 value noise 叠加（fBm）。它具备确定性和空间连续性，但低频组织、矿物团块与细脉层次有限，容易出现均匀云雾或可识别的条带。

## 当前混合场

所有场都在同一组 3D 坐标中计算，因此任意角度切割都不会像 2D 贴图那样拉伸，两半切面也会保持同源连续。

1. **宏观形体**：保留低成本 value-noise fBm，用于密度和不规则石形边界。
2. **域扭曲**：用三组不同偏移的 3D 梯度 fBm 扭曲采样坐标，破坏规则倍频与直线条带。
3. **翠色云团**：在扭曲域上取低频梯度 fBm，调制种水、颜色和明度。
4. **细脉**：对各向异性的 ridged fBm 做窄阈值整形，形成自然弯曲的絮脉，而不是正弦条纹。
5. **矿物团块**：用 3D cellular/Worley 最近特征点距离形成晶体团块，轻微抬升高水头区域的明度。
6. **微晶颗粒**：高频两层梯度 fBm 只提供小幅明度变化，避免高频噪点破坏润泽感。
7. **结构缺陷**：棉依赖细脉和独立中频场；显式裂纹仍按空间线段距离计算，因此保持稀疏、可读和与估价逻辑一致。

## 性能策略

- 纹理仍在 Web Worker 生成，不阻塞 Three.js 渲染主线程。
- 移动档使用 256×256、单层域扭曲和较少 octave；桌面档使用 384×384、双层域扭曲和更丰富细脉。
- 参考基准（Node，单线程）：移动档约 121 ms，桌面档约 291 ms；实际设备以浏览器 Worker 与 CI 指标为准。
- cellular 场只搜索相邻 27 个晶格，且所有热点采样复用对象，避免逐像素垃圾回收。

## 依据

- Ken Perlin, *Improving Noise* (SIGGRAPH 2002)：改进梯度集合与五次插值，以获得更平滑的导数和更少的视觉伪影。
- Steven Worley, *A Cellular Texture Basis Function* (SIGGRAPH 1996)：以随机特征点距离建立细胞/矿物类纹理基础。
- Inigo Quilez, *Domain Warping*：用低频噪声扰动坐标域，组合出非规则的流动结构。
